# لندینگ تعمیر چرخ خیاطی

یک وب‌سایت تک‌صفحه‌ای کاملاً استاتیک با HTML، CSS و JavaScript خالص. برای اجرای محلی یا میزبانی روی سرور، فقط Node.js لازم است و پروژه هیچ فریم‌ورک، دیتابیس یا وابستگی npm ندارد.

## ساختار

- `site/index.html`: محتوای کامل و تمام متادیتای سئو
- `site/styles.css`: طراحی، انیمیشن و حالت‌های ریسپانسیو
- `site/script.js`: منوی موبایل، عیب‌یابی تعاملی، اسلایدشو و فرم واتساپ
- `site/assets/`: تصاویر اختصاصی و بهینه‌شده
- `server.js`: سرور استاتیک Node.js با کش، فشرده‌سازی و هدرهای امنیتی
- `scripts/build.mjs`: آماده‌سازی خروجی `dist` برای انتشار

## اجرا

Node.js نسخه ۲۰ یا جدیدتر کافی است:

```bash
npm start
```

سایت به‌صورت پیش‌فرض روی `http://localhost:3000` اجرا می‌شود. برای ساخت و آزمون کامل:

```bash
npm run check
```

## وضعیت سئو پیش‌نمایش

نام کسب‌وکار، شماره تماس، شهر و ساعات کاری فعلی نمونه هستند. به همین دلیل صفحه تا زمان جایگزینی اطلاعات واقعی عمداً `noindex` است. پیش از انتشار عمومی باید اطلاعات واقعی، دامنه نهایی، تصویر Open Graph و داده ساختاریافته با هم هماهنگ و سپس مقدار robots به `index,follow` تغییر داده شود.
